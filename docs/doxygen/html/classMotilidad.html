Detailed Description

Models how cancer cells move through their microenvironment.

The Motilidad class encapsulates cell movement behaviors critical for cancer progression, particularly invasion and metastasis. It includes:

  • Basic motility properties (speed, persistence)
  • Directional bias capabilities (modeling haptotaxis/durotaxis)
  • Chemotaxis properties (response to chemical gradients)

In cancer research, altered motility is associated with:

  • Epithelial-to-mesenchymal transition (EMT)
  • Metastatic potential
  • Therapeutic resistance through cell escape
  • Collective vs. individual cell migration modes

Motilidad::Motilidad ( )

Default constructor.

Constructor initializes default motility parameters.

Initializes a cell with default motility properties:

  • Non-motile by default (es_movil = false)
  • Standard persistence time (1.0 minute)
  • Standard migration speed (1.0 μm/minute) when active
  • No initial bias or motility direction

  • Set for positive chemotaxis to first substrate by default

    Creates a motility object with parameters similar to observed cancer cell behavior:

  • Initially non-motile (es_movil = false)
  • 1.0 minute persistence time (typical for epithelial cancer cells)
  • 1.0 μm/minute migration speed (moderate migration rate, variable by cancer type)
  • No directional bias (unbiased random migration)
  • Zero motility vector (no initial movement)
  • Chemotactic response to first substrate (index 0, often oxygen)
  • Positive chemotaxis direction (cells move toward higher concentrations)

In cancer models, these parameters would be modified to reflect:

  • Highly invasive phenotypes (higher speeds, longer persistence)
  • EMT-induced motility (activation of movement)
  • Gradient-directed invasion (chemotaxis toward nutrients/away from stress)
  • Treatment effects on migration (reduced speed/persistence)

double Motilidad::bias_de_la_migracion

Strength of directional bias.

Controls how strongly the bias direction influences cell movement:

  • 0.0: No bias (pure random motion)
  • 1.0: Maximum bias (deterministic motion in bias direction) This allows modeling of cancer cells' varying responses to environmental cues.

int Motilidad::quimiotaxis_direccion

Direction of chemotactic response.

Determines whether cells move up or down the gradient:

  • 1: Positive chemotaxis (toward higher concentration)
  • -1: Negative chemotaxis (toward lower concentration) Cancer cells often show positive chemotaxis to growth factors and negative chemotaxis away from hypoxic regions.

Time duration for which a cell maintains its current direction.

Persistence time affects the pattern of cell migration:

  • Longer times create more ballistic (straight-line) movement
  • Shorter times create more diffusive (random) movement

double Motilidad::velocidad_de_migracion

Cell migration speed.

The speed at which cells migrate varies significantly across cancer types:

  • Melanoma: Often highly motile (0.1-1.0 μm/min)
