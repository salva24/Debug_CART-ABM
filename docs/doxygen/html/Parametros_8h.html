Detailed Description

Oxygen parameter definitions for cancer cell simulation.

Author
Luciana Melina Luque

Defines oxygen thresholds that govern cancer cell behaviors, particularly proliferation and necrosis. These parameters model how cancer cells respond to varying oxygen levels in the tumor microenvironment.

Inbound Dependencies: <string> Outbound Dependencies: Various cell behavior models

Usage: Instantiated by Celula and Microambiente classes to define oxygen-based responses
