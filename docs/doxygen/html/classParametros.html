Detailed Description

Parameter class defining oxygen thresholds and cell responses.

Models how cancer cells respond to different oxygen concentrations in the tumor microenvironment. Cancer cells exhibit altered metabolic responses to hypoxia compared to normal cells, including the activation of HIF-1Î± pathways, metabolic reprogramming, and resistance to hypoxia-induced death. These parameters capture key oxygen thresholds that determine cell fate decisions.

Parametros::Parametros ( )

Default constructor initializing oxygen parameters.

Constructor that initializes oxygen threshold parameters.

Sets default values for oxygen thresholds based on experimental measurements of cancer cell responses to varying oxygen levels. These defaults model general cancer cell behavior, but can be adjusted to represent specific cancer types with different hypoxia tolerance profiles.

Sets default values for oxygen-dependent behaviors based on experimental data from cancer research. These parameters model how cancer cells respond to varying oxygen levels in the tumor microenvironment, including proliferation thresholds and necrosis triggers.

Key values are set to model:

  1. Severe hypoxia (<2.5 mmHg): Maximum necrosis rate
  2. Hypoxia (2.5-5 mmHg): Increasing necrosis rate
  3. Restricted proliferation (5-10 mmHg): Cells viable but non-proliferating
  4. Optimal proliferation (>38 mmHg): Cells proliferate at maximum rate

Minimum oxygen level required for proliferation (mmHg)

Below this threshold, cancer cells cease proliferation but may remain viable. This models the phenomenon where tumor cells in hypoxic regions arrest in G0/G1 phase but can resume division when reoxygenated.

Oxygen level below which cells begin to die from necrosis (mmHg)

When oxygen drops below this threshold, cancer cells begin to undergo necrotic death. This models tumor necrotic cores, which form in regions far from blood vessels where oxygen levels are severely depleted. Cancer cells are often more resistant to hypoxia than normal cells.

Reference oxygen level for normal tissue (mmHg)
